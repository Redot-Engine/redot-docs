ImageFramesFormatLoaderExtension
================================

**Inherits:** :ref:`ImageFramesFormatLoader<class_ImageFramesFormatLoader>` **<** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Base class for creating :ref:`ImageFramesFormatLoader<class_ImageFramesFormatLoader>` extensions (adding support for extra image sequence formats).

.. rst-class:: classref-introduction-group

Description
-----------

The engine supports multiple image sequence formats out of the box, but you can choose to implement support for additional image sequence formats by extending this class.

Be sure to respect the documented return types and values. You should create an instance of it, and call :ref:`add_format_loader()<class_ImageFramesFormatLoaderExtension_method_add_format_loader>` to register that loader during the initialization phase.
